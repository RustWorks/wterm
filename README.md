## Installation

You can use the `cargo install` command:

    $ cargo install wterm

or a classic build and run:

```bash
$ git clone https://github.com/dotcypress/wterm
$ cd wterm
$ cargo build --release
$ cp target/release/wterm ~/.bin # assuming .bin is in your path
```
